Effectifs des Brigades des Maréchaussées (1750)

Source : Services historique de la défense / cote J10A (1750)

Description des colonnes :
- id : id
- ville : ville indiquée dans la source d'origine
- code : code insee des villes françaises
- ville_redressee : noms actuels des villes
- long : longitude (deg)
- lat : latitute (deg)
- province : province ou généralité d'époque
- departement : département de la prévôté
- type : prévôté générale, département de la prévôté générale ou brigade
- poste : relais de poste ou non dans la ville
- PG : nb de prévôts généraux affectés
- PP : nb de prévôts particuliers affectés
- L : nb de lieutenants affectés
- E : nb d'exempts affectés
- B : nb de brigadiers affectés
- SB : nb de sous-brigadiers affectés
- C : nb de cavaliers affectés
- T : nb de trompettes affectés
- total : nb de personnels affectés à la brigade (somme des colonnes précédentes)

Création du projet : juin 2020
